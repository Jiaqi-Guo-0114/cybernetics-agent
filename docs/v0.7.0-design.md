# v0.7.0 设计书

> 目标: 隐私安全加固 + 版本号规范化 | 原则: 安全优先

---

## 核心

移除所有文件中的真实姓名和个人邮箱信息，确保开源项目的匿名性。统一版本号管理机制，消除测试中的硬编码版本。

---

## 变更清单

### 隐私安全
- `LICENSE` — 版权持有者改为 "Cybernetics Agent Contributors"
- `README.md` / `README.en.md` / `README.es.md` / `README.fr.md` / `README.ja.md` / `README.ko.md` — 移除个人署名
- `pyproject.toml` — `authors` 字段改为匿名
- `src/cybernetics_agent/__init__.py` — `__author__` 改为 "Cybernetics Agent Contributors"

### 版本号规范
- `__init__.py` — `__version__ = "0.7.0"`
- `pyproject.toml` — `version = "0.7.0"`
- `src/cybernetics_agent/cli/dashboard.py` — 版本号同步
- `src/cybernetics_agent/cli/dashboard_fastapi.py` — 版本号同步

### 测试规范
- `tests/test_dashboard_health.py` — 从硬编码 `"0.7.0"` 改为 `from cybernetics_agent import __version__`
- 建立规范：**测试中禁止硬编码版本号**，必须动态引用 `__version__`

---
