## 3. AlertManager

统一管理规则、渠道、告警历史。

```python
class AlertManager:
    def __init__(self, config: dict):
        self._rules: list[AlertRule] = []
        self._channels: dict[str, AlertChannel] = {}
        self._history: deque[AlertEvent] = deque(maxlen=100)

    def add_rule(self, rule: AlertRule) -> None
    def register_channel(self, name: str, channel: AlertChannel) -> None
    def evaluate(self, metrics) -> list[AlertEvent]
    def get_status(self) -> dict   # 规则状态 + 历史
```

## 5. 配置热重载

```python
class ConfigWatcher:
    def __init__(self, path: str, callback: Callable, interval: float = 5.0)
    def start(self) -> None   # 启动轮询线程
    def stop(self) -> None    # 停止轮询
```

原理：每 `interval` 秒检查文件 mtime，变化时调用 callback。
不引入 watchdog，标准库 threading + os.path.getmtime 实现。
